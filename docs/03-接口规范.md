# 接口规范文档

## 1. 通用约定

### 1.1 基础信息
- 接口前缀: `/api/v1`
- 请求格式: `application/json`
- 响应格式: `application/json`
- 认证方式: `Authorization: Bearer {token}`

### 1.2 统一响应结构

```json
{
  "code": 200,
  "message": "success",
  "data": {},
  "timestamp": 1717171200000
}
```

### 1.3 状态码说明

| code | 说明 |
|------|------|
| 200 | 成功 |
| 400 | 参数错误 |
| 401 | 未登录/Token无效 |
| 403 | 权限不足 |
| 500 | 服务器内部错误 |

### 1.4 分页参数

```json
// 请求: ?page=1&size=10
// 响应 data:
{
  "records": [...],
  "total": 100,
  "size": 10,
  "current": 1,
  "pages": 10
}
```

## 2. 用户模块 `/api/v1/user`

### POST `/login/wechat` 微信登录
- **说明**: 微信一键登录，首次自动注册
- **公开接口**
- **请求**:
```json
{
  "code": "081abc123",
  "nickname": "摸鱼用户",
  "avatarUrl": "https://..."
}
```
- **响应**:
```json
{
  "token": "eyJhbGciOiJIUzI1NiJ9..."
}
```

### POST `/login/phone` 手机号登录
- **说明**: 手机号+验证码登录，需先微信注册
- **公开接口**
- **请求**:
```json
{
  "phone": "13800138000",
  "code": "123456"
}
```

### POST `/sms/send` 发送验证码
- **说明**: 发送短信验证码(演示环境固定123456)
- **公开接口**
- **请求**:
```json
{
  "phone": "13800138000"
}
```

### GET `/info` 获取用户信息
- **需登录**
- **响应**:
```json
{
  "id": 1,
  "nickname": "摸鱼用户",
  "avatarUrl": "...",
  "phone": "138****8000",
  "status": 1,
  "signInDays": 30
}
```

### POST `/sign-in` 每日签到
- **需登录**
- **响应**:
```json
{
  "points": 10
}
```

## 4. 购物车模块 `/api/v1/cart`

> 全部需登录

### GET `/list` 购物车列表
```json
{
  "items": [
    {
      "id": 1,
      "productId": 1,
      "productName": "拿铁咖啡",
      "productImage": "/images/latte.jpg",
      "quantity": 2,
      "size": "中杯",
      "temp": "热饮",
      "sugar": "标准糖",
      "price": 28.00,
      "checked": 1
    }
  ],
  "totalAmount": 56.00,
  "totalItems": 2
}
```

### POST `/add` 加入购物车
```json
{
  "productId": 1,
  "quantity": 1,
  "size": "中杯",
  "temp": "热饮",
  "sugar": "标准糖"
}
```
> 相同规格自动合并数量

### PUT `/{id}/quantity` 修改数量
- **参数**: `?quantity=3`

### PUT `/{id}/check` 切换勾选

### DELETE `/{id}` 删除购物车项

### DELETE `/clear` 清空已勾选

## 5. 订单模块 `/api/v1/order`

> 全部需登录

### POST `/submit` 提交订单
```json
{
  "payType": "WALLET",
  "couponId": 1,
  "remark": "少糖"
}
```

### POST `/{id}/pay` 发起支付
- 微信支付: 返回支付参数调起微信支付
- 储值支付: 直接扣减余额

### POST `/{id}/cancel` 取消订单
- 仅待付款状态可取消

### POST `/{id}/refund` 申请退款
- 已支付/制作中可退款

### POST `/{id}/reorder` 再来一单
- 复制订单商品生成新订单

### GET `/list` 订单列表
- **参数**: `?page=1&size=10&status=all`
- **status可选值**: all / pending / paid / preparing / ready / completed / refund

### GET `/{id}` 订单详情

    "storeName": "摸鱼咖啡（万象城店）",
