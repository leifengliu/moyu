# 摸鱼咖啡 (Moyu Coffee) — 项目概述

## 1. 项目背景

摸鱼咖啡是一款企业级咖啡店微信小程序，为顾客提供在线浏览菜单、下单购买、储值充值、积分兑换等全流程服务，覆盖咖啡店日常运营的核心场景。

## 2. 技术架构

```
┌─────────────────────────────────────────────┐
│               微信小程序客户端                 │
│         (WXML + WXSS + JavaScript)           │
└──────────────────┬──────────────────────────┘
                   │ HTTPS + JWT
┌──────────────────▼──────────────────────────┐
│           Spring Boot 3.2.x 后端             │
│  ┌─────────┬─────────┬──────────────────┐  │
│  │ Controller │ Service │ Mapper(MyBatis+)│  │
│  └─────────┴─────────┴──────────────────┘  │
└──────────────────┬──────────────────────────┘
                   │
     ┌─────────────┼─────────────┐
┌────▼────┐  ┌─────▼─────┐  ┌───▼───┐
│  MySQL  │  │   Redis   │  │ 微信API│
│  8.0+   │  │  缓存/会话  │  │       │
└─────────┘  └───────────┘  └───────┘
```

## 3. 功能模块总览

| 模块 | 功能描述 |
|------|----------|
| 用户模块 | 微信登录、手机号登录、短信验证码、用户信息、每日签到 |
| 商品模块 | 分类浏览、商品列表、商品详情(含多规格)、推荐商品、搜索 |
| 购物车模块 | 加入购物车、修改数量、勾选/取消、删除、清空 |
| 订单模块 | 提交订单、支付(微信/储值)、取消、退款、再来一单、订单追踪 |
| 储值模块 | 储值余额、充值套餐、交易明细、储值支付 |
| 积分模块 | 积分账户、积分记录、签到获取、消费获取、积分兑换 |
| 优惠券模块 | 优惠券列表、领取、使用、退还 |
| 门店模块 | 门店信息、门店列表 |

## 4. 页面路由

| 路径 | 页面 | 说明 |
|------|------|------|
| /pages/index/index | 首页 | Banner、分类、推荐商品、搜索 |
| /pages/menu/menu | 点单 | 分类Tab、商品列表、快速加购 |
| /pages/product/product | 商品详情 | 轮播图、规格选择、加购 |
| /pages/cart/cart | 购物车 | 商品列表、数量调整、优惠券、结算 |
| /pages/orders/orders | 订单列表 | Tab筛选(全部/待付款/待取餐/已完成/售后) |
| /pages/order-detail/order-detail | 订单详情 | 状态、取餐码、进度时间线、商品明细 |
| /pages/login/login | 登录 | 手机号验证码登录、微信一键登录 |
| /pages/profile/profile | 个人中心 | 用户信息、统计卡片、功能菜单 |
| /pages/wallet/wallet | 储值中心 | 余额、套餐、交易记录 |
| /pages/points/points | 积分商城 | 积分余额、兑换商品、积分明细 |

## 5. 数据流

```
用户操作 → 小程序页面 → API请求(JWT鉴权) → Spring Boot Controller
→ Service(业务逻辑) → Mapper(数据访问) → MySQL
→ 响应 JSON → 小程序渲染
```

## 7. 开发规范

- **数据库命名**: 表名按模块前缀: `sys_`, `prd_`, `ord_`, `crt_`, `wlt_`, `pts_`, `cpn_`, `cfg_`
- **API版本**: `/api/v1/`
- **统一响应**: `Result<T> { code, message, data, timestamp }`
- **金额单位**: DECIMAL(10,2), 元
- **认证方式**: JWT Bearer Token, 7天有效期
- **代码分层**: Controller → Service → Mapper
