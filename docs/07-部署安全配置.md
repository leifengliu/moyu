# 部署安全配置

## 原理

`application.yml` 中所有敏感信息已改为 `${ENV_VAR:默认值}` 格式：
- 有环境变量 → 使用环境变量
- 无环境变量 → 使用 `:` 后面的默认值（仅本地开发用）

## 服务器配置方式

在服务器上创建 `/etc/moyu-coffee.env`：

```bash
# 数据库
export DB_URL=jdbc:mysql://www.gaozhoumoyu.online:3306/moyu_coffee?useSSL=false&serverTimezone=UTC
export DB_USERNAME=moyu_prod
export DB_PASSWORD=Moyu@4321

# Redis
export REDIS_HOST=127.0.0.1
export REDIS_PORT=6379
export REDIS_PASSWORD=

# JWT (生产环境务必换成强随机字符串)
export JWT_SECRET=$(openssl rand -base64 48)

# 微信小程序
export WX_APPID=wxc453ee274ce2ddef
export WX_SECRET=40311c9840879e3303e592b9421fe4a4

# 管理后台
export ADMIN_USERNAME=admin
export ADMIN_PASSWORD=your-strong-password

# 运行环境
export SPRING_PROFILES_ACTIVE=prod
export SERVER_PORT=8080
export LOG_LEVEL=info
```

## 启动方式

### systemd 服务 (推荐)

`/etc/systemd/system/moyu-coffee.service`：

```ini
[Unit]
Description=摸鱼咖啡后端服务
After=network.target

[Service]
Type=simple
User=app
WorkingDirectory=/opt/moyu-coffee
EnvironmentFile=/etc/moyu-coffee.env
ExecStart=/usr/bin/java -jar /opt/moyu-coffee/moyu-server.jar
Restart=on-failure

[Install]
WantedBy=multi-user.target
```

启动：
```bash
sudo systemctl daemon-reload
sudo systemctl enable moyu-coffee
sudo systemctl start moyu-coffee
```

### 直接启动

```bash
source /etc/moyu-coffee.env
java -jar moyu-server.jar
```

## 本地开发

无需配置环境变量，`application.yml` 中的默认值直接可用：
- DB: localhost:3306, root/123456
- Redis: localhost:6379
- 管理后台: admin/admin123

## 安全注意事项

- `/etc/moyu-coffee.env` 权限设为 `600`（仅 owner 可读写）
- JWT_SECRET 每台机器用不同随机值
- 管理后台密码不要用默认值
- 不要把 `.env` 文件提交到 Git
