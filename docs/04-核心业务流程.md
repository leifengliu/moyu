# 核心业务流程

## 1. 用户登录流程

### 1.1 微信一键登录
```
小程序启动
  │
  ▼
wx.login() 获取临时code
  │
  ▼
POST /api/v1/user/login/wechat { code, nickname, avatarUrl }
  │
  ▼
服务端: code2session 换取 openid + unionid
  │
  ├── 新用户 → 创建 User 记录 → 创建 WalletAccount → 创建 PointsAccount
  │
  └── 老用户 → 更新 lastLoginTime, nickname, avatarUrl
  │
  ▼
生成 JWT Token (userId + openid, 7天有效期)
  │
  ▼
返回 { token } → 小程序存储到 Storage → 后续请求携带 Authorization header
```

### 1.2 手机号+验证码登录
```
用户输入手机号 → POST /api/v1/user/sms/send
  │
  ▼
Redis: SET sms:login:{phone} = "123456" EX 300
  │
  ▼
用户输入验证码 → POST /api/v1/user/login/phone { phone, code }
  │
  ▼
校验: Redis GET sms:login:{phone} == code
  │
  ▼
绑定手机号 → 返回 Token
```

## 2. 下单支付流程

```
[浏览商品] → [选择规格] → [加入购物车]
     │
     ▼
[购物车] → 勾选商品 → 选择优惠券(可选) → 去结算
     │
     ▼
POST /api/v1/order/submit { payType, couponId, remark }
     │
     ├── 校验购物车勾选项非空
     ├── 计算: totalAmount = Σ(price × qty)
     ├── 优惠计算: couponService.calculateDiscount()
     ├── payAmount = totalAmount - discount
     ├── 创建 Order (status=pending)
     ├── 创建 OrderItem[] (价格快照)
     └── 清除购物车已勾选项
     │
     ▼
[订单详情页] → 选择支付方式 → POST /api/v1/order/{id}/pay
     │
     ├── 微信支付:
     │     └── 模拟调起微信支付 → 支付成功回调 → status=paid
     │
     └── 储值支付:
           ├── 校验 wlt_account.balance >= payAmount
           ├── UPDATE wlt_account SET balance = balance - payAmount (行级锁)
           ├── INSERT wlt_transaction (type=CONSUME)
           └── status=paid
     │
     ▼
status=paid 后自动流转:
     ├── 使用优惠券: cpn_user_coupon.status=USED
     ├── 赠送积分: pts_account.balance += payAmount (每消费1元=1积分)
     └── INSERT pts_record (type=ORDER)
     │
     ▼
[商家端操作]
  paid → preparing (自动接单/店员确认)
  preparing → ready (制作完成, 生成4位取餐码)
     │
     ▼
[用户端]
  ready → completed (用户取餐/店员核销)
```

## 3. 订单状态机

```
                 ┌──────────┐
                 │  pending  │ 待付款
                 └────┬─────┘
          ┌───────────┼───────────┐
          ▼           ▼           │
    ┌──────────┐ ┌──────────┐    │
    │ cancelled│ │   paid   │    │
    │  已取消   │ │  已支付   │    │
    └──────────┘ └────┬─────┘    │
          ▲           ▼           │
          │     ┌───────────┐    │
          │     │ preparing │    │
          │     │  制作中    │    │
          │     └─────┬─────┘    │
          │           ▼           │
          │     ┌───────────┐    │
          ├─────│ refunding │◄───┘
          │     │  退款中    │
          │     └─────┬─────┘
          │           ▼
          │     ┌───────────┐
          │     │ refunded  │
          │     │  已退款    │
          │     └───────────┘
          │
          │     ┌───────────┐
          │     │   ready   │ 待取餐
          │     └─────┬─────┘
          │           ▼
          │     ┌───────────┐
          └─────│ completed │ 已完成
                └───────────┘
```

**状态转换规则**:
| 当前状态 | 可转换状态 | 操作者 |
|----------|-----------|--------|
| pending | paid | 用户(支付) |
| pending | cancelled | 用户(取消) / 系统(超时) |
| paid | preparing | 店员/自动 |
| paid | refunding | 用户(申请退款) |
| preparing | ready | 店员(制作完成) |
| preparing | refunding | 用户(申请退款) |
| ready | completed | 店员(核销取餐) |
| refunding | refunded | 系统 |

## 4. 积分规则

| 场景 | 积分变动 | 规则 |
|------|----------|------|
| 每日签到 | +10 | 每人每天限1次 |
| 消费获得 | +N | N = 实付金额(元), 向下取整 |
| 积分兑换 | -N | N = 兑换商品所需积分 |
| 退款扣回 | -N | 退回原订单获得的积分 |

## 5. 储值流程

### 5.1 充值
```
选择套餐(如: 200元送30元)
  │
  ▼
POST /api/v1/wallet/recharge { packageId: 2 }
  │
  ▼
wlt_account.balance += 230 (200+30)
wlt_account.total_recharge += 230
  │
  ▼
INSERT wlt_transaction: RECHARGE +230
```

### 5.2 消费
```
订单支付选择"储值支付"
  │
  ▼
校验: balance >= payAmount
  │
  ▼
wlt_account.balance -= payAmount (WHERE balance >= payAmount)
  │
  ▼
INSERT wlt_transaction: CONSUME -payAmount
```

### 5.3 退款
```
订单退款 → 储值支付原路退回
  │
  ▼
wlt_account.balance += payAmount
  │
  ▼
INSERT wlt_transaction: REFUND +payAmount
```

## 6. 优惠券使用流程

```
领取: POST /api/v1/coupon/receive/{couponId}
  → 校验是否已领 / 库存 → INSERT cpn_user_coupon

使用: 提交订单时传 couponId
  → calculateDiscount(): 折扣/满减计算
  → 支付成功后: status=USED

退还: 订单取消/退款
  → status 恢复为 UNUSED
```

**优惠计算规则**:
- DISCOUNT(折扣券): `discount = orderAmount × (1 - discountValue/10)`
  - 例: 8折券 → discountValue=8.00 → 折扣=订单金额×0.2
- FIXED(满减券): `discount = min(discountValue, orderAmount)`
  - 例: 满50减10 → discountValue=10.00 → 折扣=10
