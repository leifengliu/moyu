# 部署说明

## 1. 环境要求

| 组件 | 版本 | 说明 |
|------|------|------|
| JDK | 17+ | Java 运行环境 |
| MySQL | 8.0+ | 数据库 |
| Redis | 7.x | 缓存 |

## 2. 服务器环境变量配置

Spring Boot 通过 `${}` 占位符从系统环境变量读取配置（`application.yml` 中已预置默认值，仅生产环境需覆盖）。

在 `/etc/moyu/config.env` 中创建配置文件：

```bash
sudo mkdir -p /etc/moyu
sudo tee /etc/moyu/config.env << 'EOF'
# 数据库
DB_URL=jdbc:mysql://localhost:3306/moyu_coffee?useSSL=false&serverTimezone=UTC
DB_USERNAME=moyu_prod
DB_PASSWORD=你的数据库密码

# Redis
REDIS_HOST=127.0.0.1
REDIS_PORT=6379
REDIS_PASSWORD=

# JWT（生成方式: openssl rand -base64 48）
JWT_SECRET=你生成的随机字符串

# 微信小程序
WX_APPID=你的AppID
WX_SECRET=你的AppSecret

# 管理后台
ADMIN_USERNAME=admin
ADMIN_PASSWORD=你的管理员密码

# COS 对象存储
COS_SECRET_ID=你的SecretId
COS_SECRET_KEY=你的SecretKey
COS_BUCKET=你的bucket名-1234567890
COS_REGION=ap-guangzhou
COS_BASE_URL=https://你的bucket.cos.ap-guangzhou.myqcloud.com

# 运行环境
SPRING_PROFILES_ACTIVE=prod
SERVER_PORT=8080
LOG_LEVEL=info
EOF

# 设置权限（仅 root 可读，保护敏感信息）
sudo chmod 600 /etc/moyu/config.env
```

## 3. Systemd 服务配置

创建服务文件 `/etc/systemd/system/moyu.service`：

```bash
sudo tee /etc/systemd/system/moyu.service << 'EOF'
[Unit]
Description=摸鱼咖啡后端服务
After=network.target mysql.service redis.service
Wants=mysql.service redis.service

[Service]
Type=simple
User=root
WorkingDirectory=/moyu
ExecStart=/usr/bin/java -jar /moyu/moyu-server.jar
ExecStop=/bin/kill -15 $MAINPID
EnvironmentFile=/etc/moyu/config.env
Restart=on-failure
RestartSec=5

# 日志
StandardOutput=append:/moyu/logs/stdout.log
StandardError=append:/moyu/logs/stderr.log

# JVM 参数（单机低配建议）
Environment="JAVA_OPTS=-Xms256m -Xmx512m"

[Install]
WantedBy=multi-user.target
EOF

# 修改 ExecStart 为带 JVM 参数的版本
sudo sed -i 's|ExecStart=/usr/bin/java|ExecStart=/usr/bin/java $JAVA_OPTS|' /etc/systemd/system/moyu.service
```

## 4. 构建与部署

### 4.1 本地构建

```bash
cd moyu-server
mvn clean package -DskipTests
```

生成的 jar 在 `target/moyu-server-1.0.0-SNAPSHOT.jar`。

### 4.2 上传到服务器

```bash
# 上传 jar
scp target/moyu-server-1.0.0-SNAPSHOT.jar root@你的服务器IP:/moyu/moyu-server.jar

# 创建日志目录
ssh root@你的服务器IP "mkdir -p /moyu/logs"
```

### 4.3 启动服务

```bash
# 首次部署
sudo systemctl daemon-reload
sudo systemctl enable moyu
sudo systemctl start moyu

# 后续更新 jar 后只需
sudo systemctl restart moyu
```

### 4.4 常用运维命令

```bash
systemctl status moyu       # 查看服务状态
systemctl restart moyu      # 重启服务
systemctl stop moyu         # 停止服务
journalctl -u moyu -f       # 实时查看日志（systemd）
tail -f /moyu/logs/stdout.log  # 查看应用日志
```

## 5. Nginx 反向代理（推荐）

```nginx
server {
    listen 443 ssl;
    server_name www.你的域名.com;

    ssl_certificate     /etc/nginx/certs/fullchain.pem;
    ssl_certificate_key /etc/nginx/certs/privkey.pem;

    # 后端 API
    location /api/ {
        proxy_pass http://127.0.0.1:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    # COS 图片反代（遇到旧数据的 /uploads/ 路径不跨域）
    location /uploads/ {
        proxy_pass http://127.0.0.1:8080;
    }

    # 管理后台静态文件
    location /admin {
        alias /moyu/admin-front/dist;
        index index.html;
        try_files $uri $uri/ /admin/index.html;
    }
}

server {
    listen 80;
    server_name www.你的域名.com;
    return 301 https://$host$request_uri;
}
```

## 7. 目录结构（服务器上）

```
/moyu/
├── moyu-server.jar      # Spring Boot 可执行 jar
├── logs/
│   ├── stdout.log       # 应用标准输出
│   └── stderr.log       # 应用错误输出
└── uploads/             # 旧数据兼容（迁移 COS 后可删除）

/etc/
├── moyu/
│   └── config.env       # 环境变量配置（600 权限）
└── systemd/system/
    └── moyu.service     # systemd 服务定义
```

## 11. 常见问题

### Q: 数据库连接失败
检查 MySQL 是否运行、`/etc/moyu/config.env` 中账号密码是否正确。

### Q: COS 上传报错 "COS 未配置"
检查 config.env 中 COS 相关环境变量是否全部填写，SecretId/SecretKey 是否正确。

### Q: 服务启动后马上退出
```bash
journalctl -u moyu -n 50  # 查看 systemd 日志
tail -50 /moyu/logs/stderr.log  # 查看应用错误日志
```

### Q: SSL 证书怎么搞
腾讯云提供免费 SSL 证书（控制台 → SSL 证书 → 免费证书），下载 Nginx 格式，上传到服务器的 `/etc/nginx/certs/` 目录。
